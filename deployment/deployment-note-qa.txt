1. MySQL

   Installer
   - installer file

   Enable `mysql_native_password`
   ```sql
   INSTALL PLUGIN mysql_native_password SONAME 'auth_socket';
   ```

   Check plugin status
   ```sql
   SELECT plugin_name, plugin_status
   FROM information_schema.plugins
   WHERE plugin_name = 'mysql_native_password';
   ```

   Scripts
   ```sql
   -- 1. Create the schema (database)
   CREATE DATABASE IF NOT EXISTS badminton-qa;

   -- 2. Create the user
   CREATE USER IF NOT EXISTS 'badmintonqa-user01'@'localhost'
   IDENTIFIED WITH mysql_native_password BY 'user01password';

   -- 3. Grant privileges
   -- This gives the user full control only over the specific schema created above
   GRANT ALL PRIVILEGES ON badminton-qa.* TO 'badmintonqa-user01'@'localhost';

   -- 4. Apply changes
   FLUSH PRIVILEGES;

   -- 5. Verification output
   SELECT 'Summary of User Creation:' AS 'Status';
   SELECT user, host, plugin FROM mysql.user WHERE user = 'badmintonqa-user01';
   SHOW GRANTS FOR 'badmintonqa-user01'@'localhost';
   ```

   Alternative
   ```sql
   ALTER USER 'your_user'@'localhost' IDENTIFIED WITH mysql_native_password BY 'your_password';
   ```

2. Tomcat

   Package: apache-tomcat-11.0.18.zip
   - setup Java 21
   - `catalina.bat`
   - 2 packages: caulong-tc-qa, .war

   Start
   - `bin/startup.bat`

   Alternative: create a new `catalina.bat`
   ```bash
   export JAVA_HOME="/Users/yourname/path/to/tomcat/java_home"
   export JRE_HOME=$JAVA_HOME
   export CATALINA_OPTS="$CATALINA_OPTS -Dliquibase.secureParsing=false"
   ```
